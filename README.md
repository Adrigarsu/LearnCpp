# LearnCpp
 
A personal learning repository for C++ based on *A Tour of C++* (3rd edition) by Bjarne Stroustrup.
 
I'm a third-year Computer Science student with a background in Java, Spring Boot, and React, using this repo to learn modern C++ (C++20) through hands-on projects.
 
## About this repo
 
Each project targets specific chapters of the book and is designed to practice one or more core C++ concepts. Projects are built progressively — each one builds on the knowledge from the previous.

 
## Projects
 
| # | Project | Chapters | Key concepts |
|---|---------|----------|--------------|
| 1 | Expression calculator | 1 | Pointers, references, const, functions |
| 2 | Contact manager | 2 | struct, enum class, union |
| 3 | Math library | 3 | Namespaces, separate compilation, headers |
| 4 | CSV parser | 4 | Exceptions, assertions, RAII |
| 5 | Geometric shapes | 5 | Classes, inheritance, virtual, polymorphism |
| 6 | MyVector container | 6 | Rule of 5, move semantics, unique_ptr |
| 7 | Task manager | 7–9 | STL containers, algorithms, ranges |
| 8 | Mini shell | 10–12 | Concepts, variant, optional, C++20 |
 
## Reference
 
> Stroustrup, B. (2022). *A Tour of C++* (3rd ed.). Addison-Wesley.
